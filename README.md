# 📱 Smart Attendance App

##  Overview

Smart Attendance is an Android application developed as part of a Mobile Application Development (MAD) course project.
It provides a smart and efficient way to manage student attendance digitally.

##  Features

*  QR Code-based attendance system
*  View attendance records
*  Dashboard for navigation
*  Attendance tracking

##  Technologies Used

* Java (Android)
* XML (UI Design)
* Android Studio
* Gradle

##  Project Structure

* `MainActivity` – Entry screen
* `DashboardActivity` – Main dashboard
* `QRScannerActivity` – QR-based attendance
* `ViewAttendanceActivity` – Displays attendance

##  Installation

1. Clone the repository
2. Open in Android Studio
3. Build and run

##  Objective

To replace traditional attendance systems with a fast, digital, and user-friendly solution.
